# 教程简介

这是一个记录日常工作的地方。

我会把平时遇到的有趣流程、实用技巧和效率工具整理在这里，像写工作日志一样，慢慢留下自己在日常里摸索出来的东西。

## 这里会写什么

- 日常工作里的小发现
- 值得复用的工作流
- 提升效率的工具和方法
- 一些长期维护的配置文档
- 一些临时灵感和实践记录

## 写作方式

- 像随手记下的一段工作笔记，轻松一点
- 重点讲过程和思路，而不是只给结论
- 尽量真实，保留试错和调整的痕迹

如果你也喜欢记录这些琐碎但有用的流程，欢迎一起看看。
