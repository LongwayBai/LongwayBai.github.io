# 关于本站

这是一个个人技术笔记站点，用来记录日常工作中的学习心得、配置心得和效率方法。

我会把日常探索过程中积累的知识和经验整理成文档，包括工具配置、工作流优化、问题排查记录等。写作风格偏向工作笔记，重在记录思路和过程，而非仅仅给出结论。

## 内容分类

- **工具配置**：编辑器、开发环境、终端工具的配置笔记（如 LazyVim、Tmux）
- **工作流**：可复用的操作流程和效率方法
- **问题记录**：遇到的实际问题及解决方案
- **临时笔记**：尝试中的想法和阶段性总结

## 写作风格

- 记录为主，而非教学
- 侧重思路和过程，保留试错痕迹
- 随时更新，不追求完美

如果你也喜欢记录和学习，欢迎浏览交流。
